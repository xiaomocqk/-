#1. transiton 动画

transition 属性是一个简写属性

transition
{
  width: 200px;
  background: cyan;
  transition: background 800ms
}

默认值：transition: all 0 ease 0

四个过渡属性：transition-property，transition-duration，transition-timing-function，transition-delay

#2. transform 形变(translate是产生位移)。都可以进行3d变换

transform有几个属性:移动translate，缩放scale，旋转rotate，倾斜skew

translate 移动
translate
{transform: translate(20px,20px)}
scale 缩放
scale
{transform: scale(1.8, 2)}
rotate 旋转
rotate
{transform: rotate(30deg)}顺时针为正
skew 倾斜
skew
{transform: skew(10deg, 45deg)}
